Brings the power of CodeDesign.ai to WordPress, helping you create beautiful, responsive designs with ease.

== Description ==

CodeDesign.ai for WordPress connects your WordPress site with the CodeDesign.ai platform, giving you access to powerful design tools directly in your WordPress admin.

= Features =
* Seamless integration with CodeDesign.ai
* Custom full-width page template
* Easy to use settings page
* API key validation
* Automatic synchronization of designs

= How It Works =
1. Install and activate the plugin
2. Enter your CodeDesign.ai API key
3. Start creating beautiful designs directly from your WordPress admin

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/codedesign-connector` directory, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to the 'CodeDesign' menu in your admin dashboard.
4. Enter your CodeDesign.ai API key and click 'Connect'.

== Frequently Asked Questions ==

= Do I need a CodeDesign.ai account? =
Yes, you need to create an account at CodeDesign.ai to get an API key.

= How do I get an API key? =
You can obtain an API key by signing up at CodeDesign.ai.

= Can I use this plugin with any WordPress theme? =
Yes, the plugin is designed to work with any WordPress theme.

= How do I disconnect from CodeDesign.ai? =
Go to the CodeDesign settings page in your WordPress admin and click the disconnect button.
