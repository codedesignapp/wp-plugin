Brings the power of CodeDesign.ai to WordPress, helping you create beautiful, responsive designs with ease.

== Description ==

CodeDesign.ai Pages for WordPress lets you connect your [CodeDesign.ai](https://codedesign.ai/ "CodeDesign AI Website Builder") project to WordPress, giving you access to powerful design tools directly in your WordPress admin.

## Here are a few perks of publishing to Wordpress from CodeDesign: 
- Publish your website with our [AI assisted builder for WordPress](https://codedesign.ai/wordpress-ai-website-builder "CodeDesign AI Website Builder for Wordpress"), zero-coding required.
- Update the design changes on CodeDesign Builder and publish effortlessly to WordPress.
- Best of Both Worlds - Access the plugins and other tools available on WordPress while enjoying the AI perks offered by CodeDesign

##Features 
* Seamless integration with CodeDesign.ai
* Custom full-width page template
* Easy to use settings page
* API key validation
* Automatic synchronization of designs

## How It Works  
1. Install and activate the plugin on WordPress 
2. Go to CodeDesign.ai dashboard and choose a project.
3. Click on Publish button and choose **add to wordpress**.
4. Copy the provided API Key and paste it in your CodeDesign.ai API key
5. Start creating beautiful designs directly from your WordPress admin

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/codedesign-connector` directory, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to the 'CodeDesign' menu in your admin dashboard.
4. Enter your CodeDesign.ai API key and click 'Connect'.

== Frequently Asked Questions ==

= Do I need a CodeDesign.ai account? =
Yes, you need to create an account at CodeDesign.ai to get an API key.

= How do I get an API key? =
You can obtain an API key by signing up at CodeDesign.ai.

= Can I use this plugin with any WordPress theme? =
Yes, the plugin is designed to work with any WordPress theme.

= How do I disconnect from CodeDesign.ai? =
Go to the CodeDesign settings page in your WordPress admin and click the disconnect button.

== Screenshots ==

1. **Publish Panel**  
   Easily publish your CodeDesign project to WordPress using the "Add to WordPress" option from the publish panel.

2. **Connect WordPress Integration**  
   Navigate to the Integrations tab and connect your WordPress site with one click.

3. **API Key Generation**  
   Copy the generated API key from your CodeDesign dashboard to use inside the WordPress plugin.

4. **Upload Plugin**  
   Upload the CodeDesign plugin ZIP file directly from your WordPress admin panel.

5. **Activate Plugin**  
   Activate the CodeDesign plugin from the Installed Plugins list in WordPress.

6. **Enter API Key**  
   Paste the copied API key into the CodeDesign plugin settings inside WordPress and activate it.

7. **Page Synchronization**  
   Your pages, components, and functions are now synced between CodeDesign and WordPress.
